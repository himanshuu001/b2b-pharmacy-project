# B2B Pharmacy Platform

A comprehensive B2B pharmaceutical commerce and distribution platform designed to streamline interactions between distributors, retailers, hospitals, and administrators through a centralized digital ecosystem.

The platform enables efficient product management, inventory control, order processing, role-based pricing, and business operations management while maintaining enterprise-grade security and scalability.

Built for real-world pharmaceutical supply chain workflows, the platform is designed to support business growth, operational efficiency, and digital transformation across the healthcare distribution sector.

---

## Platform Overview

The B2B Pharmacy Platform serves as a unified marketplace and management solution for pharmaceutical businesses.

By connecting distributors, retailers, healthcare organizations, and administrators on a single platform, the system simplifies procurement, inventory visibility, order fulfillment, and business management processes.

The platform is designed with a modular architecture that supports scalability, maintainability, and future expansion.

## Installation

### Clone Repository

```bash
git clone https://github.com/your-username/b2b-pharmacy-platform.git
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file and configure:

```env
MONGODB_URI=
JWT_SECRET=
PORT=
CLOUDINARY_CLOUD_NAME=
CLOUDINARY_API_KEY=
CLOUDINARY_API_SECRET=
SMTP_HOST=
SMTP_PORT=
SMTP_USER=
SMTP_PASS=
```

### Run Development Server

```bash
npm run dev
```

### Run Production Server

```bash
npm start
```
